Minhas Habilidades

  • HTML

    HTML é uma linguagem de marcação para estruturar conteúdo em páginas web usando tags. Ela define a organização e exibição do conteúdo nos navegadores.

  • CSS

    CSS é uma linguagem para estilizar páginas web, controlando cores, fontes e layout. Ela complementa o HTML ao separar a apresentação do conteúdo.

  • Javascript

    Javascript é uma linguagem de programação que torna as páginas da web interativas, manipulando elementos e respondendo a eventos. Ela viabiliza aplicativos web dinâmicos e comunicação com servidores.

  • Python

    Python é uma linguagem de programação legível e versátil, aplicável em desenvolvimento web, científico e automação, com uma sintaxe clara e biblioteca abrangente.

  • SQL

    SQL é uma linguagem para gerenciar bancos de dados, permitindo a criação, consulta e modificação de dados em tabelas relacionais, essencial para aplicações que lidam com informações estruturadas.

  • Github

    GitHub é uma plataforma de código para colaboração, versionamento e compartilhamento, vital para a colaboração de desenvolvedores e gestão eficaz de código
